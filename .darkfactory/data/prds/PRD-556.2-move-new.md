## Summary

## Requirements

1. Move `cmd_new` (lines ~168–223), `_slugify`, and `_next_flat_prd_id` into `python/darkfactory/cli/new.py`.
2. Import shared helpers (`_load`, `_default_prd_dir`) from `cli._shared`.
3. Create `python/darkfactory/cli/new_test.py` with tests for `_slugify`, `_next_flat_prd_id`, and `cmd_new` argument handling.
4. Update `cli/__init__.py` to import `cmd_new` from `.new` and remove the moved code.
5. Update `cli/_parser.py` if it references `cmd_new` directly.
6. All existing tests pass.
