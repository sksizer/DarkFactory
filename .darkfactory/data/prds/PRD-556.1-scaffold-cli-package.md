# Scaffold cli/ package with __init__, _parser, _shared, and main

## Summary

Convert `python/darkfactory/cli.py` from a single module into a Python package by creating the `python/darkfactory/cli/` directory with `__init__.py`, `_parser.py`, `_shared.py`, and `main.py`. Move the shared helpers, `build_parser()`, `_configure_logging()`, `main()`, and the module-level constants into their respective new modules. `__init__.py` re-exports `main` and `build_parser` so all existing import sites continue to work unchanged.

**No subcommand functions are moved in this task.** All `cmd_*` functions remain importable from the old location via `__init__.py` re-exports. This is an empty functional change — all tests pass before and after.

## Requirements

1. Create `python/darkfactory/cli/` as a Python package.
2. Move `_find_repo_root`, `_load`, `_default_prd_dir`, `_default_workflows_dir`, `_load_workflows_or_fail`, `_action_sort_key` into `python/darkfactory/cli/_shared.py`. Also move `PRIORITY_ORDER`, `EFFORT_ORDER`, `CAPABILITY_ORDER`.
3. Create `python/darkfactory/cli/_shared_test.py` with unit tests for the shared helpers (at minimum `_find_repo_root`, `_load`, `_action_sort_key`).
4. Move `build_parser()` into `python/darkfactory/cli/_parser.py`. It imports `cmd_*` functions from `__init__.py` (or directly from submodules once they exist).
5. Move `main(argv)` and `_configure_logging(verbose)` into `python/darkfactory/cli/main.py`.
6. Create `python/darkfactory/cli/main_test.py` with a basic smoke test for the entry point.
7. `python/darkfactory/cli/__init__.py` imports from `_shared`, `_parser`, and `main`, and re-exports the public API (`main`, `build_parser`). It also holds all `cmd_*` functions temporarily (moved verbatim from old file) until later PRDs extract them.
8. Every existing `from darkfactory.cli import X` import site continues to work unchanged.
9. All existing tests pass. `just test`, `just lint`, `just typecheck`, `just format-check` all clean.

## Technical Approach

### `python/darkfactory/cli/_shared.py`

Move from the old `cli.py`:
- `PRIORITY_ORDER`, `EFFORT_ORDER`, `CAPABILITY_ORDER` — module-level dicts
- `_find_repo_root(start: Path) -> Path`
- `_default_prd_dir() -> Path`
- `_default_workflows_dir() -> Path`
- `_load_workflows_or_fail(workflows_dir: Path) -> dict[str, Workflow]`
- `_action_sort_key(prd: PRD) -> tuple[int, int, tuple[int, ...]]`
- `_load(prd_dir: Path) -> dict[str, PRD]`

### `python/darkfactory/cli/_parser.py`

Move `build_parser() -> argparse.ArgumentParser` verbatim. It references `cmd_*` functions as `set_defaults(func=cmd_X)` — these will be imported from the package's `__init__.py` initially.

### `python/darkfactory/cli/main.py`

Move:
- `_configure_logging(verbose: bool) -> None`
- `main(argv: list[str] | None = None) -> int`

### `python/darkfactory/cli/__init__.py`

```python
"""CLI package — re-export hub.

Holds cmd_* functions temporarily until extracted by subsequent PRDs.
Re-exports main and build_parser for backwards compatibility.
"""
from darkfactory.cli._shared import (
    PRIORITY_ORDER, EFFORT_ORDER, CAPABILITY_ORDER,
    _find_repo_root, _default_prd_dir, _default_workflows_dir,
    _load_workflows_or_fail, _action_sort_key, _load,
)
from darkfactory.cli._parser import build_parser
from darkfactory.cli.main import main

# All cmd_* functions live here temporarily...

__all__ = ["main", "build_parser"]
```

### Migration strategy

1. Delete `python/darkfactory/cli.py`.
2. Create `python/darkfactory/cli/` directory.
3. Create `_shared.py`, `_parser.py`, `main.py`, `__init__.py`.
4. `__init__.py` initially holds all `cmd_*` implementations (moved verbatim) plus imports from internal modules.
5. Verify all tests pass — this is a no-op refactor at this point.
