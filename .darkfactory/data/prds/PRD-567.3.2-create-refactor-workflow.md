# Create refactor/cleanup workflow with broad file-operation permissions

## Summary

PRDs that primarily involve deleting, moving, or reorganizing code need broader file-operation permissions than the standard task workflow. This task creates a `refactor` workflow with permissions for `rm`, `mv`, `find`, and `git rm`, matched by a predicate that targets PRDs tagged `refactor` or `cleanup`, or with "delete"/"remove" in the title.

## Requirements

1. Create `src/darkfactory/workflows/refactor/workflow.py` using `PRD_IMPLEMENTATION_TEMPLATE.compose()`.
2. Predicate: matches PRDs with `kind == "task"`, `status == "ready"`, and either:
   - `"refactor"` or `"cleanup"` in `prd.tags`, or
   - `"delete"` or `"remove"` in `prd.title.lower()`.
3. Priority: 4 (above task at 3, below planning at 5).
4. Tool list extends the task workflow with:
   - `Bash(rm:*)` — direct file deletion
   - `Bash(mv:*)` — file moves/renames
   - `Bash(find:*)` — find files for bulk operations
   - `Bash(git rm:*)` — git-tracked deletion
   - `Bash(git mv:*)` — git-tracked moves
5. Create prompt files (`role.md`, `task.md`, `verify.md`) — can start as copies of the task workflow prompts with an added note about deletion/move permissions being available.
6. Add `__init__.py` to the workflow directory.

## Technical approach

```python
# src/darkfactory/workflows/refactor/workflow.py
from darkfactory.templates_builtin import PRD_IMPLEMENTATION_TEMPLATE
from darkfactory.workflow import AgentTask, ShellTask


def _is_refactor_or_cleanup(prd, prds):
    if prd.kind != "task" or prd.status != "ready":
        return False
    if {"refactor", "cleanup"} & set(prd.tags):
        return True
    title_lower = prd.title.lower()
    return "delete" in title_lower or "remove" in title_lower


workflow = PRD_IMPLEMENTATION_TEMPLATE.compose(
    name="refactor",
    description=(
        "Refactoring/cleanup workflow — broader file-operation permissions "
        "for PRDs that reorganize, move, or delete code."
    ),
    applies_to=_is_refactor_or_cleanup,
    priority=4,
    middle=[
        AgentTask(
            name="implement",
            prompts=["prompts/role.md", "prompts/task.md"],
            tools=[
                "Read", "Edit", "Write", "Glob", "Grep",
                "Bash(cargo:*)", "Bash(pnpm:*)", "Bash(just:*)", "Bash(uv:*)",
                "Bash(git add:*)", "Bash(git rm:*)", "Bash(git mv:*)",
                "Bash(git status:*)", "Bash(git diff:*)", "Bash(git log:*)",
                "Bash(rm:*)", "Bash(mv:*)", "Bash(find:*)",
            ],
            model_from_capability=True,
            retries=1,
            verify_prompts=["prompts/verify.md"],
        ),
        ShellTask("test", cmd="just test", on_failure="retry_agent"),
        ShellTask("format", cmd="just format", on_failure="fail"),
        ShellTask("lint", cmd="just lint format-check", on_failure="retry_agent"),
        ShellTask("typecheck", cmd="just typecheck", on_failure="retry_agent"),
    ],
)
```
