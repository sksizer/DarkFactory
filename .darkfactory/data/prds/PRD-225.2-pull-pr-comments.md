# Pull unaddressed PR comments via gh

## Summary

Create a module that fetches all unaddressed review comments from a GitHub PR using the `gh` CLI. This module handles parsing the JSON output from `gh pr view`, filtering by resolved status, author, and timestamp, and returning structured comment data that can be composed into a feedback prompt.

## Requirements

1. Create `python/darkfactory/pr_comments.py` with a `fetch_pr_comments(pr_number, *, filters) -> list[ReviewThread]` function.
2. Fetch issue comments, review comments (line-anchored), and review summaries via `gh pr view <num> --json comments,reviews,reviewThreads`.
3. Filter out resolved threads by default; include them when `--all` is passed.
4. Filter out comments posted before a given commit timestamp when `--since <commit>` is used.
5. Filter to a specific reviewer when `--reviewer <user>` is used.
6. Support `--from-pr-comment <id>` to fetch a single specific comment.
7. Filter out comments authored by the harness bot (identified by a configurable bot username or `[harness]` prefix) to avoid feedback loops.
8. Return structured `ReviewThread` dataclasses with: author, path, line, body, thread_id, is_resolved, posted_at, and replies (list of comment bodies in the thread).

@dataclass
class CommentFilters:
    include_resolved: bool = False

def fetch_pr_comments(
    pr_number: int,
    filters: CommentFilters | None = None,
) -> list[ReviewThread]:
    """Fetch and filter PR review threads from GitHub."""
    raw = _gh_fetch(pr_number)

### Timestamp resolution for `--since`

When `--since <commit>` is provided, resolve the commit SHA to a timestamp via `git log -1 --format=%aI <commit>` and filter out comments posted before that timestamp.
