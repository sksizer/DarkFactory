# User config directory (~/.config/darkfactory/)

## Summary

Create a `paths.py` module that resolves the user-level config directory (`~/.config/darkfactory/`) honoring `$XDG_CONFIG_HOME`. This module provides the canonical path functions used by the cascade resolvers (PRD-222.5, PRD-222.6) to discover user-level workflows and config. The workflows subdirectory is created on first access if missing; `config.toml` is lazy (absent means no user overrides).

## Requirements

1. Create `python/darkfactory/paths.py` with functions to resolve user-level paths:
   - `user_config_dir() -> Path` — returns `$XDG_CONFIG_HOME/darkfactory/` or `~/.config/darkfactory/`.
   - `user_config_file() -> Path | None` — returns the path to `config.toml` if it exists, else `None`.
   - `user_workflows_dir() -> Path` — returns the workflows subdirectory, creating it if absent.
2. Honor `$XDG_CONFIG_HOME` if set; otherwise default to `~/.config/`.
3. The `workflows/` subdirectory is created on first access (so users can drop files in without manually creating the dir).
4. `config.toml` is never auto-created — its absence simply means "no user-level overrides."
5. Platform-appropriate: on macOS use `~/.config/` (XDG is common there too), not `~/Library/`. Keep it simple and consistent with the XDG convention.

## Technical Approach

### `python/darkfactory/paths.py`

```python
from pathlib import Path
import os

def user_config_dir() -> Path:
    """Return the user-level darkfactory config directory."""
    xdg = os.environ.get("XDG_CONFIG_HOME")
    base = Path(xdg) if xdg else Path.home() / ".config"
    return base / "darkfactory"

def user_config_file() -> Path | None:
    """Return path to user config.toml if it exists."""
    f = user_config_dir() / "config.toml"
    return f if f.is_file() else None

def user_workflows_dir() -> Path:
    """Return user workflows dir, creating if absent."""
    d = user_config_dir() / "workflows"
    d.mkdir(parents=True, exist_ok=True)
    return d
```

### `tests/test_paths.py`

- Test `XDG_CONFIG_HOME` override.
- Test `user_config_file()` returns `None` when file doesn't exist.
- Test `user_workflows_dir()` creates the directory.
