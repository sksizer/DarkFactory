# Template data-flow contracts (provides/expects)

## Summary

Extend `WorkflowTemplate` with `provides` and `expects` fields that declare
what `PhaseState` bundles the open phase will produce and what the close phase
requires. The unified engine asserts these contracts at phase boundaries,
giving clear error messages when a required data bundle is missing.

## Requirements

1. Add `provides: frozenset[type]` and `expects: frozenset[type]` fields to
   `WorkflowTemplate` with empty frozenset defaults.
2. After the open phase completes, the engine asserts that all `provides`
   types are present in `PhaseState`.
3. Before the close phase runs, the engine asserts that all `expects` types
   are present in `PhaseState`.
4. Assertion failures produce clear error messages naming the missing bundle
   type.
5. Update `PRD_IMPLEMENTATION_TEMPLATE` in `templates_builtin.py` to declare
   its contracts (e.g., open phase provides `PrdContext`).
6. Add unit tests for the assertion logic.
7. mypy strict passes.

## Technical Approach

### `src/darkfactory/templates.py`

```python
@dataclass(frozen=True)
class WorkflowTemplate:
    # ... existing fields ...
    provides: frozenset[type] = frozenset()
    expects: frozenset[type] = frozenset()
```

### Engine assertions

In the unified engine, after open phase tasks complete:

```python
for required_type in template.provides:
    if not ctx.state.has(required_type):
        raise RuntimeError(
            f"Open phase failed to provide {required_type.__name__} "
            f"(declared in template '{template.name}')"
        )
```

Before close phase tasks run:

```python
for required_type in template.expects:
    if not ctx.state.has(required_type):
        raise RuntimeError(
            f"Close phase requires {required_type.__name__} but it is "
            f"not in PhaseState (declared in template '{template.name}')"
        )
```

### Template declarations

```python
PRD_IMPLEMENTATION_TEMPLATE = WorkflowTemplate(
    # ... existing fields ...
    provides=frozenset({PrdContext}),
    expects=frozenset({PrdContext}),
)
```
