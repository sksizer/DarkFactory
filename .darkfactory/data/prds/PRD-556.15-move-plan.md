# Move cmd_plan into cli/plan.py with colocated tests

## Summary

Extract `cmd_plan` and its helpers (`_describe_task`, `_resolve_base_ref`, `_check_runnable`) from `cli/__init__.py` into `python/darkfactory/cli/plan.py`. This is one of the larger subcommands (~200 lines with helpers). Create colocated `plan_test.py`.

## Requirements

1. Move `_describe_task` (~901–922), `_resolve_base_ref` (~923–980), `_check_runnable` (~981–1008), and `cmd_plan` (~1009–1090) into `python/darkfactory/cli/plan.py`.
2. Import shared helpers from `cli._shared` and other modules as needed.
3. If `_resolve_base_ref` or `_check_runnable` are also needed by `cmd_run`, promote them to `cli._shared` or import from `plan.py` in `run.py`.
4. Create `python/darkfactory/cli/plan_test.py` with tests for `_describe_task`, `_resolve_base_ref`, `_check_runnable`, and plan output formatting.
5. Update `cli/__init__.py` and remove the moved code.
6. All existing tests pass.
