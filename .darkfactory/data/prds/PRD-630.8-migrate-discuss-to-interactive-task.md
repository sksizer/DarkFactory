# Migrate discuss operation to InteractiveTask

## Summary

Replace the `discuss_prd` builtin (which internally calls `spawn_claude()`)
with an `InteractiveTask` in the discuss operation's task list. Wire the
unified engine to dispatch `InteractiveTask` to `spawn_claude()`. Remove
the `discuss_prd` builtin.

## Requirements

1. Add `InteractiveTask` dispatch to the unified engine: when the engine
   encounters an `InteractiveTask`, it calls `spawn_claude()` with the
   task's prompt file, tools, model, and effort level.
2. Update the discuss operation in `commands/discuss/operation.py` to use
   `InteractiveTask` instead of `BuiltIn("discuss_prd")`.
3. Similarly update the critique task if it exists as a builtin.
4. Remove the `discuss_prd` builtin from `builtins/discuss_prd.py` (or
   gut it to a stub if other code references it).
5. Update `cli/discuss.py` if it directly references the builtin.
6. All existing tests pass.

## Technical Approach

### Engine dispatch

In `run_tasks()`, add handling for `InteractiveTask`:

```python
elif isinstance(task, InteractiveTask):
    spawn_claude(
        prompt_file=task.prompt_file,
        tools=task.tools,
        model=task.model or default_model,
        effort_level=task.effort_level,
        cwd=ctx.cwd,
    )
```

### Discuss operation update

```python
# commands/discuss/operation.py
tasks=[
    BuiltIn("gather_prd_context"),
    InteractiveTask(
        name="discuss",
        prompt_file="prompts/discuss.md",
        effort_level="max",
    ),
    InteractiveTask(
        name="critique",
        prompt_file="prompts/critique.md",
        effort_level="max",
    ),
    BuiltIn("commit_prd_changes"),
]
```

### Builtin removal

Delete or empty `src/darkfactory/builtins/discuss_prd.py`. Remove its
registration from the builtin registry.
