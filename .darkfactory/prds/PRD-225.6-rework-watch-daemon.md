# Local polling rework-watch daemon

## Summary

A long-running local process `prd rework-watch` that polls open PRs for new review comments and triggers `prd rework <PRD-X>` automatically when unaddressed feedback appears. This runs on the user's hardware (not GitHub Actions), polling at a configurable interval.

This is a phase-2 piece — it depends on the manual `prd rework` command being solid first.

## Requirements

1. Create `src/darkfactory/rework_watch.py` with the polling loop and state management.
2. Register `prd rework-watch` subcommand with sub-commands: default (foreground), `--daemon`, `--status`, `--pause`, `--resume`, `--stop`.
3. Every N seconds (default 60, configurable via `--interval`), list open PRs whose head branch matches `prd/PRD-*`.
4. For each PR, fetch comments and compare against the last-seen comment ID stored in `.darkfactory/state/rework-watch.json`.
5. If new unresolved comments exist, invoke `prd rework PRD-X` as a subprocess.
6. Update the last-seen pointer after each successful check.
7. Safety: refuse to start if worktree for any active PR is missing.
8. Safety: maximum N rework cycles per PR per hour (default 3) to prevent thrashing.
9. Safety: honor the per-PRD process lock from PRD-217 so manual `prd run` doesn't race with the watcher.
10. A pause file at `.darkfactory/state/rework-watch.pause` halts the watcher without stopping the process.

## Technical Approach

### `src/darkfactory/rework_watch.py`

```python
@dataclass
class WatchState:
    last_seen: dict[str, str]  # prd_id -> last comment ID
    rework_counts: dict[str, list[float]]  # prd_id -> timestamps of recent reworks
    last_poll: float | None

def poll_once(state: WatchState) -> WatchState:
    """Single poll iteration: check all open PRs for new comments."""
    prs = _list_open_prd_prs()
    for pr in prs:
        prd_id = _extract_prd_id(pr.branch)
        if _is_paused():
            continue
        if _rate_limited(state, prd_id):
            continue
        threads = fetch_pr_comments(pr.number)
        new_threads = _filter_new(threads, state.last_seen.get(prd_id))
        if new_threads:
            _invoke_rework(prd_id)
            state = _update_state(state, prd_id, threads)
    return state

def run_watcher(interval: int = 60):
    """Main polling loop."""
    state = _load_state()
    while True:
        if not _is_paused():
            state = poll_once(state)
            _save_state(state)
        time.sleep(interval)
```

### CLI registration

```python
p_watch = sub.add_parser("rework-watch", help="Poll PRs for new review comments")
p_watch.add_argument("--interval", type=int, default=60, help="Poll interval in seconds")
p_watch.add_argument("--daemon", action="store_true", help="Run in background")
p_watch.add_argument("--status", action="store_true", help="Show watcher status")
p_watch.add_argument("--pause", action="store_true", help="Pause the watcher")
p_watch.add_argument("--resume", action="store_true", help="Resume the watcher")
p_watch.add_argument("--stop", action="store_true", help="Stop the daemon")
p_watch.set_defaults(func=cmd_rework_watch)
```

### Rate limiting

Track timestamps of rework invocations per PRD. Before invoking, check how many occurred in the last hour. If >= max (default 3), skip and log a warning.

### `tests/test_rework_watch.py`
