# Move cmd_cleanup into cli/cleanup.py with colocated tests

## Summary

Extract `cmd_cleanup` and its helper functions (`_remove_worktree`, `_find_worktree_for_prd`, `_cleanup_single`, `_cleanup_merged`, `_cleanup_all`) from `cli/__init__.py` into `src/darkfactory/cli/cleanup.py`. Create colocated `cleanup_test.py`.

## Requirements

1. Move `cmd_cleanup` (lines ~418–436) and helpers `_remove_worktree` (~287–300), `_find_worktree_for_prd` (~301–325), `_cleanup_single` (~326–339), `_cleanup_merged` (~340–359), `_cleanup_all` (~360–417) into `src/darkfactory/cli/cleanup.py`.
2. Import shared helpers from `cli._shared` and `darkfactory.checks` as needed.
3. Create `src/darkfactory/cli/cleanup_test.py` with tests for the cleanup helpers.
4. Update `cli/__init__.py` to import `cmd_cleanup` from `.cleanup` and remove the moved code.
5. All existing tests pass.
