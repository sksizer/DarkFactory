# Move summarize_agent_run to its own submodule with colocated tests

## Summary

Extract the `summarize_agent_run` builtin function and its exclusively-used private helpers (`_format_tool_counts`, `_format_invocations`) from `builtins/__init__.py` into `src/darkfactory/builtins/summarize_agent_run.py`. Create a colocated `summarize_agent_run_test.py` with unit tests. Per the parent PRD's open question, audit existing test coverage and backfill as needed.

## Requirements

1. Create `src/darkfactory/builtins/summarize_agent_run.py` containing the `summarize_agent_run` function decorated with `@builtin("summarize_agent_run")`, plus its private helpers: `_format_tool_counts`, `_format_invocations`.
2. Create `src/darkfactory/builtins/summarize_agent_run_test.py` with unit tests covering: no `last_invoke_result` (early return), successful summary generation, tool count formatting, invocation count formatting.
3. Remove `summarize_agent_run`, `_format_tool_counts`, and `_format_invocations` from `builtins/__init__.py`.
4. Add submodule import in `__init__.py` to trigger registration.
5. All existing tests pass.

## Technical Approach

### `src/darkfactory/builtins/summarize_agent_run.py`

```python
from __future__ import annotations
from darkfactory.builtins._registry import builtin
from darkfactory.workflow import ExecutionContext

def _format_tool_counts(tool_counts: dict[str, int]) -> str: ...
def _format_invocations(ctx: ExecutionContext) -> str: ...

@builtin("summarize_agent_run")
def summarize_agent_run(ctx: ExecutionContext) -> None: ...
```

### `src/darkfactory/builtins/summarize_agent_run_test.py`

- `last_invoke_result is None` → early return, `ctx.run_summary` unchanged
- Successful run → `ctx.run_summary` set with expected markdown format
- `_format_tool_counts({})` → `"none"`
- `_format_tool_counts({"Read": 5, "Edit": 3})` → `"Edit×3, Read×5"` (sorted)
- `_format_invocations` with counts 0, 1, N
