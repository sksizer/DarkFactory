# Planning workflow as a system operation

## Summary

Re-cast the planning/decomposition workflow as a `SystemOperation` with `accepts_target=True`. Planning takes one PRD as input and creates many child PRDs as output — it's the same shape as other system operations (operates on the PRD set, not implementing a single PRD), and it doesn't fit the "one branch per PRD" `Workflow` model.

Invoked via `prd system run plan --target PRD-X --execute`.

We will keep the original planning workflow as well, to support inline planning during DAG based execution.

## Requirements

1. The operation must accept a `--target PRD-X` argument identifying the epic or feature PRD to decompose.
2. The operation creates a dedicated branch (e.g. `system/plan-PRD-X-{date}`) and worktree for the decomposition work.
3. An agent task reads the target PRD and produces child PRD files in the worktree.
4. A shell task runs `prd validate` to verify the child PRDs are well-formed.
5. If validation fails, the agent is retried once with the validation errors.
6. On success, the operation commits, pushes, and creates a PR with the new child PRDs.
7. The operation sets `creates_pr=True` and `accepts_target=True`.
8. The original planning workflow continues to work as before

## Technical Approach

**New directory**: `.darkfactory/operations/plan/`

**New file**: `.darkfactory/operations/plan/operation.py`

```python
from darkfactory.system import SystemOperation
from darkfactory.workflow import AgentTask, BuiltIn, ShellTask

operation = SystemOperation(
    name="plan",
    description=(
        "Decompose an epic or feature PRD into implementable child task PRDs. "
        "Takes a single target PRD as input, produces N child PRDs as output."
    ),
    creates_pr=True,
    accepts_target=True,
    pr_title="chore(prd): decompose {target_prd}",
    pr_body=(
        "Auto-generated by `prd system run plan --target {target_prd}`.\n\n"
        "Creates child PRDs for decomposition of {target_prd}."
    ),
    tasks=[
        BuiltIn("ensure_worktree"),
        AgentTask(
            name="decompose",
            prompts=["prompts/decompose.md"],
            tools=["Read", "Write", "Glob", "Grep", "Bash(uv run prd validate:*)"],
            model="opus",
            model_from_capability=False,
            retries=1,
            verify_prompts=[],
            sentinel_success="PRD_EXECUTE_OK",
            sentinel_failure="PRD_EXECUTE_FAILED",
        ),
        ShellTask(
            name="validate",
            cmd="uv run prd validate",
            on_failure="retry_agent",
        ),
        BuiltIn("commit", kwargs={"message": "chore(prd): {target_prd} decomposition"}),
        BuiltIn("push_branch"),
        BuiltIn("create_pr"),
    ],
)
```

**New file**: `.darkfactory/operations/plan/prompts/decompose.md` — the decomposition prompt template with placeholders for the target PRD path, the PRD format guide, and acceptance criteria.

**New file**: `tests/test_plan_operation.py` — tests covering:
- Operation loads and has `accepts_target=True`
- Operation is rejected without `--target`
- Task list is well-formed
