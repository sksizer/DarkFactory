# Add help text examples to top 5 commands

## Summary

No subcommand has `epilog` or usage examples in its help text. Modern CLIs (`gh`, `cargo`, `uv`) all provide examples. Add epilog examples to the 5 most-used commands: `run`, `plan`, `new`, `status`, `cleanup`.

## Technical approach

Add `epilog` and `formatter_class=argparse.RawDescriptionHelpFormatter` to each subparser in `_parser.py`. Example for `run`:

```python
sub_run = sub.add_parser(
    "run",
    help="Run a workflow against a PRD (dry-run unless --execute)",
    epilog=textwrap.dedent("""\
        examples:
          prd run PRD-042                         # dry-run single PRD
          prd run PRD-042 --execute               # execute single PRD
          prd run PRD-500 --execute --max-runs 3  # run epic, cap at 3
          prd run --all --execute                  # drain ready queue
          prd run --all --priority high --execute  # drain high-priority only
    """),
    formatter_class=argparse.RawDescriptionHelpFormatter,
)
```

Also update the top-level parser description from "Pumice PRD harness CLI" to "DarkFactory: PRD lifecycle management" and add a brief getting-started hint.

Commands to add examples for:
1. `run` -- most complex, most flags
2. `plan` -- common dry-run workflow
3. `new` -- creation workflow
4. `status` -- first command new users try
5. `cleanup` -- has subtle flag interactions
