# Define Finding dataclass and detector registry

## Summary

Create the foundational data model and extension infrastructure for the
transcript analyzer. This includes the `Finding` dataclass that all detectors
produce and the decorator-based detector registry that mirrors the existing
`BUILTINS` registry shape in `_registry.py`.

## Requirements

1. Define a `Finding` dataclass with fields: `category` (str), `severity`
   (literal: `info`, `warning`, `error`), `message` (str), and optional
   `line` (int | None) pointing to the transcript line.
2. Create a `DETECTORS` registry dict mapping detector names to callables
   with signature `(events: list[dict]) -> list[Finding]`.
3. Provide a `@detector("name")` decorator that registers a function into
   `DETECTORS`, rejecting duplicate names (mirroring `@builtin` behavior).
4. Export `Finding`, `DETECTORS`, and the `detector` decorator from the
   module.

## Technical Approach

### File: `src/darkfactory/builtins/analyze_transcript_detectors.py`

1. Define `Finding` as a `@dataclass` (or `@dataclass(frozen=True)` for
   immutability). Fields:
   - `category: str`
   - `severity: Literal["info", "warning", "error"]`
   - `message: str`
   - `line: int | None = None`
2. Create module-level `DETECTORS: dict[str, DetectorFunc]` where
   `DetectorFunc = Callable[[list[dict]], list[Finding]]`.
3. Implement `detector(name: str)` decorator:
   ```python
   def detector(name: str):
       def wrapper(func):
           if name in DETECTORS:
               raise ValueError(f"Duplicate detector: {name}")
           DETECTORS[name] = func
           return func
       return wrapper
   ```
4. Reference `src/darkfactory/builtins/_registry.py` for the existing
   pattern to mirror.

## Acceptance Criteria

- [ ] AC-1: `Finding` dataclass is defined with all four fields and correct
      types.
- [ ] AC-2: `DETECTORS` dict is importable and starts empty.
- [ ] AC-3: `@detector("name")` registers a function and rejects duplicates
      with a clear error.
- [ ] AC-4: Module is importable without side effects (no detectors
      registered at import time — those come in PRD-559.2).
