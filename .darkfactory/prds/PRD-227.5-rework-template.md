# Define REWORK_TEMPLATE

## Summary

Define `REWORK_TEMPLATE` for the rework/revision workflow that PRD-225 introduces. The rework template differs from implementation: it opens by checking that a PR exists, resuming the worktree, and fetching review comments; it closes by committing and pushing to the existing branch (no `create_pr`). This template enforces rework-specific invariants by construction.

## Requirements

1. Define `REWORK_TEMPLATE` in `src/darkfactory/templates_builtin.py`.
2. Opening tasks:
   - `check_pr_exists` — assert the PR is open and fetch its URL
   - `resume_worktree` — resume (not create) the existing worktree
   - `fetch_review_comments` — pull PR review comments for context
   - `set_status(to="in-progress")`
3. Closing tasks:
   - `summarize_agent_run`
   - `commit_transcript`
   - `set_status(to="review")`
   - `commit(message="chore(prd): {prd_id} address review feedback")`
   - `push_branch` (no `create_pr` — PR already exists)
4. Middle requires at least 1 `AgentTask`.
5. Add tests for the rework template's compose/validation behavior.

## Technical Approach

### `src/darkfactory/templates_builtin.py`

```python
REWORK_TEMPLATE = WorkflowTemplate(
    name="rework",
    description="Rework lifecycle: resume existing PR, address feedback, push updates.",
    open=[
        BuiltIn("check_pr_exists"),
        BuiltIn("resume_worktree"),
        BuiltIn("fetch_review_comments"),
        BuiltIn("set_status", kwargs={"to": "in-progress"}),
    ],
    middle_kinds=[AgentTask, ShellTask],
    middle_required={
        AgentTask: (1, None),
    },
    close=[
        BuiltIn("summarize_agent_run"),
        BuiltIn("commit_transcript"),
        BuiltIn("set_status", kwargs={"to": "review"}),
        BuiltIn("commit", kwargs={"message": "chore(prd): {prd_id} address review feedback"}),
        BuiltIn("push_branch"),
    ],
)
```

### Tests

- Valid rework composition produces correct task order
- Missing `AgentTask` in middle raises `TemplateViolation`
- Closing does not include `create_pr`
