# Final cleanup: delete old monolith files and verify package

## Summary

Delete `tests/test_builtins.py` (its coverage has been migrated into colocated test files by PRD-549.3–549.11). Verify that `src/darkfactory/builtins/__init__.py` is a clean re-export hub with no builtin implementations remaining. Confirm that the `builtins/` package is the single source of truth and all shared helpers in `_shared.py` are properly factored.

## Requirements

1. Delete `tests/test_builtins.py` — all its test coverage must already exist in colocated `*_test.py` files.
2. Verify `builtins/__init__.py` contains only imports and re-exports, no function implementations.
3. Verify `_shared.py` contains only helpers used by 2+ builtins; helpers used by exactly one builtin live in that builtin's submodule.
4. Grep the codebase for any remaining `from darkfactory.builtins import` sites and confirm they all resolve.
5. All tests pass. `just test`, `just lint`, `just typecheck`, `just format-check` all clean.

## Technical Approach

### Delete old test file

```bash
git rm tests/test_builtins.py
```

### Audit `__init__.py`

Ensure it only contains:
- Imports from `_registry` (`BUILTINS`, `BuiltInFunc`, `builtin`)
- Submodule imports to trigger registration
- `__all__` re-exports
- No function bodies

### Audit `_shared.py`

Verify each helper in `_shared.py` is imported by 2+ submodules. If any helper is used by only one, move it into that submodule.

### Import site verification

```bash
grep -rn "from darkfactory.builtins import" src/ tests/ workflows/
grep -rn "from darkfactory import builtins" src/ tests/ workflows/
```

Confirm all import sites resolve without error.

## Acceptance Criteria

- [ ] AC-1: `tests/test_builtins.py` is deleted.
- [ ] AC-2: `builtins/__init__.py` contains only imports and re-exports — no builtin implementations.
- [ ] AC-3: Every helper in `_shared.py` is used by 2+ builtins.
- [ ] AC-4: All `from darkfactory.builtins import X` sites in the codebase resolve.
- [ ] AC-5: All tests pass. `just test`, `just lint`, `just typecheck`, `just format-check` all clean.
- [ ] AC-6: No dead code remains from the old monolith.
