## Prerequisites

- **Python 3.12+** installed
- **Git** installed and configured
- **uv** or **pipx** for Python tool installation (recommended: [uv](https://docs.astral.sh/uv/))
- **Claude Code** CLI installed (for agent-powered workflows)
