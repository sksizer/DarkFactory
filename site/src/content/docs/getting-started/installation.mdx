import { Tabs, TabItem, Steps, Aside } from '@astrojs/starlight/components';

<Steps>

1. **Install DarkFactory**

   <Tabs syncKey="installer">
     <TabItem label="uv (recommended)">
       ```bash frame="terminal"
       uv tool install darkfactory
       ```
     </TabItem>
     <TabItem label="pipx">
       ```bash frame="terminal"
       pipx install darkfactory
       ```
     </TabItem>
     <TabItem label="pip">
       ```bash frame="terminal"
       pip install darkfactory
       ```
     </TabItem>
   </Tabs>

   Verify the installation:

   ```bash frame="terminal"
   $ prd --version
   darkfactory 0.1.0
   ```

2. **Initialize your project**

   Navigate to your project root and run:

   ```bash frame="terminal"
   $ prd init
   Created .darkfactory/
   Created .darkfactory/data/prds/
   Created .darkfactory/config.toml
   ```

   This creates the `.darkfactory/` directory where PRDs, workflows, and runtime state live.

3. **Check the project layout**

   ```
   your-project/
   ├── .darkfactory/
   │   ├── data/
   │   │   ├── prds/       # Your PRD files (tracked in git)
   │   │   └── archive/    # Archived PRDs (tracked in git)
   │   ├── workflows/      # Custom workflows (optional, tracked)
   │   ├── config.toml     # Project config (optional, tracked)
   │   ├── transcripts/    # Agent transcripts (optional, tracked)
   │   └── events/         # Event logs (git-ignored)
   ├── .worktrees/         # Git worktrees (git-ignored, at repo root)
   ├── .harness-transcripts/ # Raw agent output (git-ignored, at repo root)
   └── ... your code ...
   ```

</Steps>

<Aside type="tip">
  Add `.worktrees/`, `.harness-transcripts/`, and `.darkfactory/events/` to your repo-root `.gitignore`. Track `data/prds/`, `data/archive/`, `workflows/`, and `config.toml` under `.darkfactory/`. The `prd init` command sets this up automatically.
</Aside>
