# ensure_worktree refuses to resume on stale state

## Summary

Add a safety contract to `ensure_worktree`: before taking the resume path, check whether the PR for this branch is already merged or closed, and whether the local branch has diverged from origin. If the branch's PR is resolved, refuse with a clear error directing the user to `prd cleanup`. If origin has diverged, warn loudly.

This is a `[safety contract]` — hard-coded into the primitive because bypassing it silently corrupts state.

## Requirements

1. Extract the check logic into `darkfactory.checks.is_resume_safe(branch, repo_root) -> ResumeStatus` so other tools (`prd cleanup`, `prd reconcile`, tests) can call the same logic.
2. `ResumeStatus` is a dataclass/enum indicating: `safe`, `pr_merged`, `pr_closed`, `diverged_from_origin`.
3. `ensure_worktree` BuiltIn calls `is_resume_safe` when it detects an existing worktree. On non-safe result, raise with a clear error message.
4. PR state is checked via `gh pr list --head <branch> --state all --json state,mergedAt`. If `gh` is not installed or not auth'd, log a warning and proceed with local-only checks (branch existence). Don't hard-fail on missing `gh`.
5. Divergence check: compare local branch tip with `origin/<branch>`. If origin has commits the local doesn't, warn.

## Technical Approach

### `src/darkfactory/checks.py`

```python
@dataclass
class ResumeStatus:
    safe: bool
    reason: str  # human-readable explanation
    kind: str    # "safe" | "pr_merged" | "pr_closed" | "diverged"

def is_resume_safe(branch: str, repo_root: Path) -> ResumeStatus:
    """Check if resuming work on branch is safe."""
    # 1. Check PR state via gh (with fallback)
    try:
        result = subprocess.run(
            ["gh", "pr", "list", "--head", branch, "--state", "all", "--json", "state,mergedAt"],
            capture_output=True, text=True, cwd=repo_root,
        )
        if result.returncode == 0:
            prs = json.loads(result.stdout)
            for pr in prs:
                if pr["state"] == "MERGED":
                    return ResumeStatus(safe=False, reason=f"PR for {branch} is merged; run `prd cleanup` to start fresh", kind="pr_merged")
                if pr["state"] == "CLOSED":
                    return ResumeStatus(safe=False, reason=f"PR for {branch} is closed; run `prd cleanup` to start fresh", kind="pr_closed")
    except FileNotFoundError:
        logger.warning("gh not found; skipping PR state check")

    # 2. Check local vs origin divergence
    # ...compare refs...

    return ResumeStatus(safe=True, reason="", kind="safe")
```

### `src/darkfactory/builtins.py`

In `ensure_worktree`, when the resume path is detected:

```python
from darkfactory.checks import is_resume_safe
status = is_resume_safe(branch, repo_root)
if not status.safe:
    raise HarnessError(status.reason)
```
