# Move ensure_worktree to its own submodule with colocated tests

## Summary

Extract the `ensure_worktree` builtin function and its exclusively-used private helpers (`_worktree_target`, `_branch_exists_local`, `_branch_exists_remote`) from `builtins/__init__.py` into `src/darkfactory/builtins/ensure_worktree.py`. Create a colocated `ensure_worktree_test.py` with unit tests covering the function's branches.

## Requirements

1. Create `src/darkfactory/builtins/ensure_worktree.py` containing the `ensure_worktree` function decorated with `@builtin("ensure_worktree")`, plus its private helpers: `_worktree_target`, `_branch_exists_local`, `_branch_exists_remote`.
2. Create `src/darkfactory/builtins/ensure_worktree_test.py` with unit tests covering: dry-run path, resume existing worktree, branch-exists error, successful creation, lock acquisition and timeout.
3. Remove `ensure_worktree` and its private helpers from `builtins/__init__.py`.
4. Add `from darkfactory.builtins import ensure_worktree as ensure_worktree_mod` (or equivalent) import in `__init__.py` to trigger registration.
5. All existing tests pass. `just test`, `just lint`, `just typecheck`, `just format-check` all clean.

## Technical Approach

### `src/darkfactory/builtins/ensure_worktree.py`

```python
from __future__ import annotations
import logging
import subprocess
from pathlib import Path
from filelock import FileLock, Timeout
from darkfactory.builtins._registry import builtin
from darkfactory.workflow import ExecutionContext

_log = logging.getLogger(__name__)

def _worktree_target(ctx: ExecutionContext) -> Path: ...
def _branch_exists_local(repo_root: Path, branch: str) -> bool: ...
def _branch_exists_remote(repo_root: Path, branch: str) -> bool: ...

@builtin("ensure_worktree")
def ensure_worktree(ctx: ExecutionContext) -> None: ...
```

### `src/darkfactory/builtins/ensure_worktree_test.py`

Unit tests using mocked subprocess and mocked `ExecutionContext`:
- Dry-run sets `ctx.worktree_path` and `ctx.cwd` without subprocess calls
- Existing worktree path resumes without creating a new one
- Branch exists locally/remotely raises `RuntimeError`
- Successful creation calls `git worktree add`
- Lock timeout raises with clear message

### `src/darkfactory/builtins/__init__.py`

Add import: `from darkfactory.builtins import ensure_worktree as _ensure_worktree_mod`
Keep re-export: `from darkfactory.builtins.ensure_worktree import ensure_worktree`
